# 대화형 통신 편집기에서 UI 연결

>[!NOTE]
>
> 대화형 통신 기능은 얼리어답터 프로그램에서 사용할 수 있습니다. 액세스 권한을 요청하려면 회사 이메일 주소를 통해 `aem-forms-ea@adobe.com`으로 이메일을 보내시기 바랍니다.

**UI 연결**&#x200B;은(는) 대화형 통신(IC) 편집기 위에 빌드된 전문적이고 단순화된 인터페이스입니다. 현장 동료 및 서비스 에이전트와 같은 고객 응대 전문가가 라이브 상호 작용 중 실시간으로 개인화되고, 순응적이며, 정확한 커뮤니케이션을 생성하도록 설계되었습니다.

![IC 문서 찾기](/help/forms/interactive-communication/assets/associate-ui-preview.png)

## UI 인터페이스 연결

연결 UI는 빠르고 안전한 통신 생성을 가능하게 하는 깔끔한 2개의 패널 작업 공간을 제공합니다.

### 왼쪽 패널: 데이터 입력

- Associates가 고객별 정보를 입력하거나 확인합니다.
- 유효성 검사, 도우미 텍스트 및 필수 필드는 정확한 입력을 안내합니다.

### 오른쪽 패널: 실시간 미리 보기

- 최종 문서의 미리 보기를 즉시 표시합니다.
- 연결된 사용자가 데이터를 채우면 자동으로 업데이트됩니다.

### 인스턴트 문서 생성

- 완료된 커뮤니케이션을 생성하거나 다운로드합니다.

## 사용자 담당자 및 책임

Associate UI는 각각 고유한 책임이 있는 세 개의 핵심 역할로 구성됩니다.

### &#x200B;1. 관리자

시스템 설정, 거버넌스, 백엔드 통합 및 사용자 액세스를 담당합니다.

| 책임 | 초점 |
|---------------|-------|
| 시스템 구성 | 핵심 인프라, 사용자 그룹, 양식 데이터 모델(FDM), 출력 을 설정합니다. |
| 거버넌스 및 보안 | 사용자 권한을 관리하고 시스템 규정 준수를 보장합니다. |
| 통합 관리 | 백엔드 통합 및 라이브 고객 데이터 연결을 유지 관리합니다. |

### &#x200B;2. 작성자

연결 UI를 사용하여 대화형 통신을 디자인하고 관리합니다. ß

| 책임 | 초점 |
|---------------|-------|
| IC 제작 및 디자인 | 레이아웃, 브랜딩 및 규격 문서 구조를 만듭니다. |
| 필드 구성 | 데이터 필드를 매핑하고, 편집 가능, 필수 및 읽기 전용 필드를 정의합니다. |
| 게시 및 지원 | IC를 게시하고 연결 액세스에 대한 링크를 공유합니다. |

### &#x200B;3. 연결

연결 UI를 사용하여 고객을 지원하고 정보를 업데이트하며 준수 커뮤니케이션을 생성합니다.


| 책임 | 초점 |
|---------------|-------|
| 데이터 확인 | 왼쪽 입력 패널을 통해 고객 데이터를 채우거나 확인합니다. |
| 미리 보기 및 유효성 검사 | 실시간 미리 보기 패널을 사용하여 정확성을 보장합니다. |
| 게재 | PDF/이메일을 생성하고 승인된 채널을 통해 전송합니다. |

>[!NOTE]
>
> 연결은 **forms-associates** 그룹의 일부여야 합니다.

## 동적 사용 사례

연결 UI는 실시간 서비스 요구 사항이 있는 산업에 중요한 즉각적인 개인화된 문서 생성을 지원합니다.

| 업계 | 예시 사용 사례 |
|----------|-------------------|
| **금융 서비스** | 실시간 대출 확인서, 위험 프로필 요약, 계정 생성 등을 생성합니다. |
| **보험** | 즉시 보험 증빙 카드 또는 청구서 처리 요약 작성 |
| **헬스 케어** | 계산된 급여 및 일정을 사용하여 환자 치료 계획 요약을 만듭니다. |
| **공공 부문** | 경찰 확인 보고서, 시민 서비스 영수증, 고충확인서, 사례 업데이트 요약 등을 현장에서 생성합니다. |
| **정부** | 복지 제도 등록을 위한 신청 상태 요약, 서비스 승인서 및 실시간 커뮤니케이션을 생성합니다. |

## UI 연결 워크플로 활성화

작성자는 아래 단계에 따라 UI 액세스 연결을 위해 대화형 통신(IC)을 구성하고 게시할 수 있습니다.

>[!NOTE]
>
> 연결에 지원되는 구성 요소: 날짜 필드, 숫자 필드, 텍스트 필드.

### IC 만들기

인터랙티브 커뮤니케이션을 디자인 및 구성하여 브랜딩, 데이터 바인딩, 규정 준수 규칙 및 통합이 올바르게 설정되도록 합니다.

### 연결 UI 활성화

상단 작업 표시줄에서 Associate UI 옵션을 활성화하여 Associate-Driven에 IC를 사용할 수 있도록 합니다.

### 구성 요소에서 UI 연결 활성화

### 편집 가능한 필드 구성

필수 필드 섹션에서 연결할 수 있는 필드를 활성화합니다.
정확하고 통제된 데이터 입력을 보장하기 위해 유효성 검사를 설정합니다.

### IC 게시

모든 구성을 완료한 후 보안 액세스를 위해 대화형 통신을 게시합니다.

### 게시된 IC를 Associates와 공유

게시된 IC 링크를 연결 대상에 제공하여 인증하고, 고객별 정보를 입력하고, 유효한 입력으로 최종 통신을 생성할 수 있습니다.

**연결 UI**&#x200B;은(는) 구조화된 콘텐츠 작성과 실시간 고객 참여 간의 차이를 줄여줍니다.\
직관적인 디자인, 강력한 백엔드 구성 및 엄격한 규정 준수 제어를 결합하여 조직은 규모에 맞게 **빠르고 정확하며 개인화된 커뮤니케이션**&#x200B;을 제공할 수 있습니다.
