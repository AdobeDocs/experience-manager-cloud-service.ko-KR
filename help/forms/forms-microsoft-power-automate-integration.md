# Microsoft Power와 적응형 양식 통합 자동화 {#architecture}

AEM Workflows 또는 Microsoft Power 자동화와 같은 비즈니스 프로세스 관리 솔루션을 사용하여 비즈니스 프로세스를 자동화할 수 있습니다. 제출 시 고급 자동화 클라우드 흐름을 실행하도록 적응형 양식을 구성할 수 있습니다. 제출된 데이터, 첨부 파일 및 기록 문서가 Power Automatic Cloud 플로우에 전송됩니다.

응용 Forms 편집기에서는 다음을 제공합니다. **Microsoft 전원 자동화 흐름 호출** 적응형 양식 데이터, 첨부 파일 및 기록 문서를 전송하는 제출 작업은 Power Automate Cloud Flow로 전송됩니다.
