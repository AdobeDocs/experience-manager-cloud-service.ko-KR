# Go Live {#golive-migration}

AEM as a Cloud Service에서 유연하고 성공적인 go-live를 수행하려면 다음 단계를 실행하는 것이 좋습니다.

* 코드 및 컨텐츠 고정 기간 예약
* 최종 컨텐츠 추가 수행
* 전체 테스트 반복
* 성능 및 보안 테스트 실행
* 잘라내기
