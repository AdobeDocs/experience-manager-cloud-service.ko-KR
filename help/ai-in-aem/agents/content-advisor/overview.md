# 콘텐츠 관리자 에이전트 {#content-advisor}

콘텐츠 어드바이저 에이전트는 콘텐츠 검색 및 최적화에 숨겨진 방대한 운영 작업을 흡수하여 진화하는 콘텐츠에서 발생하는 마찰을 제거하여 팀이 오늘날의 AI 시대에 보다 빠르고 안정적으로 변경 사항을 제공할 수 있도록 지원합니다.

## Jobs {#jobs}

Content Advisor 에이전트는 많은 시간과 많은 양의 작업을 자동화합니다.

* [컨텐츠 검색](#discovery)
* [콘텐츠 최적화](#optimization)

콘텐츠 어드바이저 에이전트는 팀이 수동적이고 긴 프로세스를 최신 일관되게 유지하는 AI 지원 워크플로우로 전환하여 비즈니스가 목표를 달성하는 데 도움이 됩니다.

>[!IMPORTANT]
>
>AI가 생성한 응답은 정확하지 않거나 오해의 소지가 있을 수 있습니다. 제안된 수정 사항과 응답을 다시 확인하십시오.
>

### 콘텐츠 검색 작업 {#discovery}

[콘텐츠 검색 작업](/help/ai-in-aem/agents/content-advisor/discovery.md)은(는) 사용자 의도를 해석하고, AEM에서 가장 관련성이 높은 승인된 콘텐츠를 검색하여 재사용할 수 있도록 표시하므로 적절한 자산을 적절한 순간에 연결합니다.

### 콘텐츠 최적화 작업 {#optimization}

[콘텐츠 최적화 작업](/help/ai-in-aem/agents/content-advisor/content-optimization.md)은(는) 다양한 채널에 대한 시각적, 이미지 및 비디오를 개인화하고 조정함으로써 경험 수명 주기 전반에 걸쳐 참여와 전환을 향상시킵니다.
