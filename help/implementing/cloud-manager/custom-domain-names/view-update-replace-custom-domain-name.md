# 사용자 지정 도메인 이름 {#view-update-replace-cdn} 보기 및 업데이트 및 바꾸기

보기 및 업데이트 메뉴를 사용하여 하나 이상의 사용자 지정 도메인 이름에 대한 세부 사항을 볼 수 있습니다.
또는 사용자 지정 도메인 이름에 사용할 SSL 인증서를 변경할 수 있습니다.

>[!NOTE]
>SSL 인증서는 유효해야 하며 사용자 지정 도메인 이름을 포함해야 합니다.
