# 부여된 액세스 권한 {#access-rights-granted}

Adobe은 IMS(Adobe Identity Management 시스템)에서 회사의 **조직** 식별자를 만듭니다. 이 ID를 통해 모든 사용자와 해당 권한을 관리할 수 있습니다. 이 조직의 구성원이 되어야 하고 [!UICONTROL Experience Cloud] 서비스에 대한 액세스 권한이 부여될 각 사용자는 자신의 **Adobe ID**&#x200B;을(를) 가져야 합니다.

## 사용자 ID 유형 {#user-identity-types}

Adobe ID을 시작하려면 사용 가능한 ID 유형 중 하나를 사용하여 Adobe ID을 얻는 방법에 대한 자세한 지침을 보려면 [Adobe ID 유형 관리](https://helpx.adobe.com/enterprise/using/identity.html)를 방문하십시오.

## 사용자 및 역할 {#users-and-roles}

Adobe이 회사의 조직을 만들었으면 지정된 관리자가 이 조직의 첫 번째 구성원으로 추가됩니다. 관리자는 기본적으로 관리자 권한을 부여받고 [!UICONTROL AEM Managed Services] **Product** 및 하나 이상의 [!UICONTROL 클라우드 관리자] **제품 프로필**&#x200B;을(를) 할당합니다. Admin Console을 사용하여 팀 사용자를 구성하고 관리하는 방법에 대한 자세한 내용은 [사용자 및 역할 추가](add-users-roles.md)를 방문하십시오.

이러한 권한이 부여된 상태에서 관리자는 이제 단일 사인온(Adobe ID 사용)으로 설정하여 [!UICONTROL Experience Cloud] 서비스에 액세스하고, AEM 클라우드 환경에 로그인하고 [!UICONTROL 클라우드 관리자]를 사용합니다.
