# Adobe ID {#adobe-id} 다운로드


Adobe ID이 있어야 합니다. Cloud Service 사용자는 작성자 계층에 액세스하려면 Adobe ID를 사용해야 합니다. 이제 IMS(Adobe Identity Management System)와 통합되어 단일 사인온 기능을 제공합니다.

사용 가능한 ID 유형 중 하나를 사용하여 Adobe ID을 얻는 방법에 대한 자세한 지침은 [Adobe ID 유형 관리](https://helpx.adobe.com/enterprise/admin-guide.html/enterprise/using/identity.ug.html)를 참조하십시오.
