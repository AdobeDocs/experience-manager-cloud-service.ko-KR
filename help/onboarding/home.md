* 클라우드 서비스로서의 Experience Manager 액세스를 위한 요구 사항
* 특정 기능의 가용성을 제어하는 사용자 역할 이해
* Cloud Manager git 저장소 설정
* 클라우드 서비스로서의 Experience Manager 액세스
* 데이터 보호 및 개인 정보 보호 규정의 기초 사항
