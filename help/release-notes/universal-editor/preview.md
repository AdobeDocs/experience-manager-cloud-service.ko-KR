# 범용 편집기 미리보기 릴리스 정보 {#preview}

다음은 범용 편집기 **미리보기 버전**&#x200B;에 대한 릴리스 정보입니다. 이러한 기능은 현재 범용 편집기의 **미리보기 환경**&#x200B;에서 사용할 수 있습니다. 이러한 기능은 2025년 12월 11일에 일반 출시될 예정입니다.

이 **미리보기** 릴리스 정보는 범용 편집기에 대한 예정된 변경 사항을 알 수 있도록 편의를 위해 제공되며, [미리보기 환경으로 전환](/help/sites-cloud/authoring/universal-editor/navigation.md#user-properties)하여 테스트할 수 있습니다.

>[!TIP]
>
>범용 편집기의 **현재 릴리스 정보**&#x200B;에 대해 알아보려면 [범용 편집기 릴리스 정보](/help/release-notes/universal-editor/current.md) 문서를 참조하십시오.

>[!NOTE]
>
>실제 릴리스의 콘텐츠와 릴리스 날짜는 변경될 수 있습니다.

## 예정된 새로운 기능 {#what-is-new}

* 새로운 리치 텍스트 편집기의 기존 표에 지원이 추가되었습니다.
* 탭 키는 새 리치 텍스트 편집기의 중첩 목록에 대해 활성화됩니다.
* 메타 태그 `dev-login`을(를) 통해 개발자 로그인 기능을 사용하지 않도록 설정할 수 있습니다.
* 이제 오버레이 섹션을 마우스 오른쪽 버튼으로 클릭하면 상황별 메뉴가 표시됩니다.
* 이제 범위 들여쓰기가 새 리치 텍스트 편집기에서 지원됩니다
* 콘텐츠 조각에 대해 약식 복사가 구현되었습니다.

## 예정된 개선 사항 {#other-improvements}

* 이제 다중 필드가 컨텍스트에서 변경되면 속성 레일이 동기화됩니다.
* 이제 콘텐츠 조각 선택기가 AEM 6.5 인스턴스에서 예상대로 열립니다.
* 이제 Esc 키를 누르면 새 리치 텍스트 편집기의 대화 상자가 닫힙니다.
* 이제 구성 요소를 선택한 경우에만 **구성 요소 제거** 작업을 사용할 수 있습니다.
* 이제 사용된 인스턴스를 기반으로 올바른(이전 또는 새) 콘텐츠 조각 편집기가 열립니다(호스트 이름이 AEM as a Cloud Service 패턴인 경우 새 편집기를 사용하고 그렇지 않으면 기존 편집기를 사용).
* 중복 작업에 필터 유효성 검사가 추가됩니다.
* 이제 속성 레일에서 긴 제목이 잘립니다.
* 이제 값이 10개가 넘는 다중 사이트 관리자 배열이 제대로 처리됩니다.
* 이름이 같은 여러 구성 요소를 만들 때 발생하는 충돌 오류가 이제 제대로 처리됩니다.
* 값이 10보다 큰 다중 사이트 관리자 배열 처리가 추가되었습니다.
