# Setting Up Dynamic Media {#setting-up-dynamic-media}

[Dynamic Media](https://www.adobe.com/solutions/web-experience-management/dynamic-media.html) 는 웹, 모바일 및 소셜 사이트에서 사용할 수 있도록 자동으로 크기가 조정되는 풍부한 시각적 머천다이징 및 마케팅 자산을 On-Demand로 전달하여 자산을 관리하는 데 도움이 됩니다. 마스터 자산 세트를 사용하면 Dynamic Media는 글로벌, 확장 가능 및 성능 최적화 네트워크를 통해 실시간으로 다양한 유형의 풍부한 컨텐츠를 생성하고 전달합니다.
